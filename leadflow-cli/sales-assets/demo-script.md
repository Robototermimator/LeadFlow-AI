# Demo Video Script (3–5 minutes)

1. Open terminal and run `leadflow --help`.
2. Explain that the tool works offline and needs no API.
3. Run a live command:
   `leadflow generate --niche "dentists" --location "Cape Town" --service "website redesign" --count 10 --mode deep --output leads.csv`
4. Show progress indicators and preview in terminal.
5. Open CSV file and point out lead scores, detected problems, and suggested offers.
6. Highlight outreach and follow-up emails in output rows.
7. Close with CTA:
   "I can deliver qualified leads + outreach campaigns weekly using this system."
