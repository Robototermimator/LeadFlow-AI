1. Open LeadFlow AI home page and explain value proposition.
2. Login with demo account.
3. Go to Generate Leads.
4. Enter niche: Dentists, location: Cape Town, offer: Website Redesign, keywords: outdated website, poor SEO, no booking system.
5. Generate 10 leads and highlight demo mode if no API key is set.
6. Open Leads page, filter by uncontacted, sort by score.
7. Copy outreach and follow-up emails for a top lead.
8. Export CSV and show download.
